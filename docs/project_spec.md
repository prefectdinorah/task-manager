# Task Manager - Спецификация

## Обзор

Современный Task Manager с минималистичным дизайном в стиле Claude.ai.

## Цели

1. Удобный инструмент для задач
2. Современные веб-технологии
3. Workflow с AI-агентом
4. Освоить Supabase + Vercel

## Целевая аудитория

- Разработчики
- Любители минимализма
- Пример modern stack

## MVP функции

### 1. Управление задачами
- Создание
- Редактирование
- Удаление
- Отметка выполнения

### 2. Фильтрация
- Все / Активные / Завершённые

### 3. UI/UX
- Минималистичный дизайн
- Плавные анимации
- Адаптивный
- Тёмная тема

## Post-MVP

- Приоритеты, категории
- Дедлайны, подзадачи
- Аутентификация
- Совместная работа
- Поиск, архив, статистика

## Метрики

- UI < 100ms
- Lighthouse > 90
- Test coverage > 70%
