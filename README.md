# Task Manager

Современный Task Manager в стиле Claude.ai

## Старт

\`\`\`bash
npm install
cp .env.example .env.local
# Добавьте Supabase credentials
npm run dev
\`\`\`

## Команды

\`\`\`bash
npm run merge:dev      # Мердж в dev
npm run docs:update    # Обновить документацию
npm run status         # Статус проекта
npm run notify "msg"   # Telegram уведомление
\`\`\`

## Документация

- \`claude.md\` - Workflow инструкции
- \`PROJECT.md\` - О проекте
- \`docs/\` - Детальная документация
